# LGBTQ+ pride-flag attributions

All 27 SVG files in this directory were downloaded from Wikimedia Commons via
`https://commons.wikimedia.org/wiki/Special:FilePath/<File>.svg` by
`tools/fetch_pride_flags.sh`. Re-run that script if any file is replaced
upstream.

## Licensing

Most pride flags are simple horizontal stripe designs and are widely considered
**below the threshold of originality** (PD-shape) in many jurisdictions, so the
underlying flag designs are typically free of copyright. The specific SVG
renderings on Commons carry per-file licenses — predominantly CC0, PD-shape,
or a permissive CC-BY/CC-BY-SA. Each Commons source page (linked below) has
the authoritative license + author info; spot-check before re-using in a
context where attribution is required.

For practical use in Flag Game (in-app quiz display), every file in this set
is freely licensable.

## Judgment calls on flags with multiple designs

A few identities have more than one widely-used flag design. Choices made:

- **Lesbian** — used the 2018 Emily Gwen 7-stripe "sunset" design
  (`Lesbian_pride_flag_2018.svg`). This is the current community consensus,
  having largely replaced both the 2010 "lipstick" version (which has
  associations with its creator that many in the community reject) and the
  older labrys design. Notably, the 5-stripe "community" variant is also seen
  in the wild but the 7-stripe is the more recognised SVG on Commons.

- **Polyamory** — used the 2022 Red Howell "infinity heart" redesign
  (`Polyamory_flag_with_infinity_heart.svg`) rather than the original 1995
  Jim Evans pi-symbol tricolor. The infinity-heart design was created
  specifically to replace the older version (which used Pythagorean symbolism
  that the community had outgrown) and has become the dominant modern flag.

- **Two-Spirit** — used `Two-Spirit_Flag.svg` (the 6-colour rainbow with two
  feathers and a circle). Several Two-Spirit flag designs exist; this is the
  most widely circulated SVG version on Commons. As a flag representing
  Indigenous LGBTQ+ identities, designs may vary by community and nation —
  this is one common visual reference, not a universal symbol.

- **Queer** — used `Queer_flag.svg` (a lavender / pink / blue gradient
  design). There is no single agreed "queer" flag; multiple competing designs
  exist (some intentionally avoid stripes; some use a black background).
  This was the most prominent SVG result on Commons.

- **Demigender** — there is **no canonical SVG on Wikimedia Commons for the
  umbrella "demigender" identity** (Commons has specific files for demigirl,
  demiboy, and demi-agender, but not a generic demigender). The file used
  here (`Demiagender_Flag.svg`, slug `demigender.svg`) is the demi-agender
  variant — it shares the same gray-and-yellow "demi-" visual idiom and is
  the closest available SVG. Players seeing this flag will recognise the
  demi-* pattern.
