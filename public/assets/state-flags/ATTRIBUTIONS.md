# US state + DC flag attributions

All 51 SVG files in this directory were downloaded from Wikimedia Commons via
`https://commons.wikimedia.org/wiki/Special:FilePath/Flag_of_<State>.svg` by
`tools/fetch_state_flags_retry.sh`. Re-run that script if any file is replaced
upstream.

Every US state flag is an official state insignia. Per Wikimedia Commons,
**every file in this set is freely licensed for any use** — virtually all are
released as Public Domain (PD-USGov-state, PD-shape, or explicit author-PD
dedication) by the publishing state authority. A small number additionally
carry a CC license fallback on Commons; in those cases the underlying state
emblem is still PD and the redrawn SVG is freely licensed.

License details for each file are kept on its Commons source page (link below).
Suggested attribution if needed: *"Flag of <State>, Wikimedia Commons"*.
